# Docs Folder

This folder is reserved for project documentation only.

Rules:
- Frontend web app source lives only in `web/`
- GitHub Pages is deployed from the `web/` folder through `.github/workflows/deploy-pages.yml`
- Do not place webpage runtime files such as `index.html`, app JS, or app CSS in `docs/`

Current documentation examples:
- `master_codex.md`
- `memory_onboard.md`
- `communication.md`
- `backend_hosting_options.md`
